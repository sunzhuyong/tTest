# 人脸识别系统 - 规范文档

## 1. 项目概述

- **项目名称**: FaceRecognitionDemo
- **项目类型**: Windows桌面应用程序 (WinForms + OpenCV)
- **核心功能**: 实时人脸检测、人脸数据采集与注册、人脸数据导出到Excel
- **目标用户**: 需要进行人脸数据管理的用户

## 2. UI/UX 规范

### 2.1 布局结构

**主窗口布局**:
- 顶部: 工具栏区域 (高度: 60px)
- 左侧: 视频预览区域 (宽度: 640px, 高度: 480px)
- 右侧: 功能面板 (宽度: 300px)
  - 人脸列表区域
  - 操作按钮区域
- 底部: 状态栏 (高度: 30px)

### 2.2 视觉设计

**配色方案**:
- 主色调: #2196F3 (蓝色)
- 次要色: #FFFFFF (白色)
- 背景色: #F5F5F5 (浅灰)
- 强调色: #4CAF50 (绿色 - 识别成功), #F44336 (红色 - 错误)
- 文字颜色: #333333 (深灰)

**字体**:
- 标题: Microsoft YaHei UI, 14pt, 粗体
- 正文: Microsoft YaHei UI, 10pt, 常规
- 按钮: Microsoft YaHei UI, 10pt, 常规

**间距**:
- 组件间距: 10px
- 内边距: 15px

### 2.3 组件设计

**按钮样式**:
- 正常状态: 背景 #2196F3, 文字白色, 圆角 4px
- 悬停状态: 背景 #1976D2
- 禁用状态: 背景 #BDBDBD

**视频预览框**:
- 边框: 2px solid #2196F3
- 圆角: 8px
- 背景: 黑色

**人脸列表**:
- 每项高度: 50px
- 选中背景: #E3F2FD
- 包含缩略图 + 姓名 + 注册时间

## 3. 功能规范

### 3.1 核心功能

1. **摄像头预览**
   - 实时显示摄像头捕获的视频
   - 支持摄像头设备选择
   - 帧率: 30fps

2. **人脸检测**
   - 使用OpenCV Haar Cascade进行人脸检测
   - 检测到人脸时在视频中绘制绿色矩形框
   - 支持同时检测多个人脸

3. **人脸注册**
   - 点击"注册人脸"按钮开始采集
   - 自动捕获人脸图像
   - 输入姓名并保存
   - 存储到本地SQLite数据库

4. **人脸识别**
   - 实时比对人脸与已注册人脸
   - 显示匹配结果（姓名/未知）

5. **Excel导出**
   - 导出所有已注册人脸信息
   - 包含字段: ID, 姓名, 注册时间, 照片路径
   - 用户通过SaveFileDialog选择保存路径

### 3.2 用户交互流程

1. 启动程序 → 自动打开默认摄像头
2. 点击"开始识别" → 实时检测人脸
3. 检测到人脸后点击"注册" → 弹出输入框 → 输入姓名 → 保存
4. 点击"导出Excel" → 选择保存路径 → 导出完成

### 3.3 数据存储

- **数据库**: SQLite (本地文件)
- **照片存储**: 本地文件夹 `Faces/`
- **Excel格式**: .xlsx

## 4. 技术栈

- **框架**: .NET Framework 4.7.2 / .NET 6.0
- **UI**: WinForms
- **OpenCV**: OpenCvSharp4
- **数据库**: SQLite (Microsoft.Data.Sqlite)
- **Excel**: ClosedXML
