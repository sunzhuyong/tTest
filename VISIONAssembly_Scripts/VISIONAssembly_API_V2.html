GvTool - 工具参数操作

功能：用于和工具参数交互。禁止自己定义，直接使用内置的 GvTool 对象。

方法列表

  • GetToolData(str, is_raw_string) - 获取工具参数
  • SetToolData(str, value, is_raw_string) - 设置工具参数

GetToolData(str, is_raw_string)

参数 类型 说明
str string 工具参数名，格式："工具名.参数名"，如 "找圆工具_001.圆心"
is_raw_string bool 可选，默认 False。True=获取原始字符串(bytes)，False=UTF-8转换

返回值：对应工具参数的拷贝

注意：获取的是一份拷贝，对拷贝修改不会自动同步到工具，需要配合 SetToolData() 使用。

示例代码

# 获取圆心坐标（分量的方式）
x = GvTool.GetToolData("找圆工具_001.圆心.X")
y = GvTool.GetToolData("找圆工具_001.圆心.Y")
r = GvTool.GetToolData("找圆工具_001.半径")

# 获取整体对象
center = GvTool.GetToolData("找圆工具_001.圆心")
print(center.GetX(), center.GetY())

SetToolData(str, value, is_raw_string)

参数 类型 说明
str string 工具参数名，格式："工具名.参数名"
value any 要设置的值
is_raw_string bool 可选，默认 False。True=设置原始字符串(bytes)，False=GBK转换

返回值：bool，表示是否设置成功

示例代码

# 设置工具参数
GvTool.SetToolData("找圆工具_001.半径", 50.0)
GvTool.SetToolData("Blob工具_001.总像素数上限", GvVisionAssembly.Limit_Int(1000))

GvVar - 变量管理

功能：用于操作变量管理中的变量。禁止自己定义，直接使用内置的 GvVar 对象。

方法列表

  • GetVar(str, is_raw_string) - 获取变量
  • SetVar(str, value, is_raw_string) - 设置变量

GetVar(str, is_raw_string)

参数 类型 说明
str string 变量名，如 "#L1"、"@G1"
is_raw_string bool 可选，默认 False

返回值：变量的拷贝

示例代码

# 获取变量
a = GvVar.GetVar("#L0")
x = GvVar.GetVar("#X")
y = GvVar.GetVar("#Y")
pos_i = GvVar.GetVar("#i")

SetVar(str, value, is_raw_string)

参数 类型 说明
str string 变量名，如 "#L1"、"@G1"
value any 要设置的值

返回值：bool，表示是否设置成功

示例代码

# 设置变量
GvVar.SetVar("#L0", 100)
GvVar.SetVar("#Count", 5)

GvVisionAssembly - 核心功能模块

功能：提供 GUI 创建、消息报告、上下限设置等核心功能。

构造函数/方法列表

  • Limit_Int(值) - 创建整型上下限
  • Limit_Double(值) - 创建浮点型上下限
  • GsScriptGuiStyle() - 创建 GUI 样式
  • GsScriptGuiText() - 创建文本
  • GsScriptGuiRect() - 创建矩形
  • GsScriptGuiCircle() - 创建圆形
  • GsScriptGuiLine() - 创建直线
  • GsScriptGuiCross() - 创建十字
  • GcScriptGuiArray() - 创建 GUI 数组
  • sc2Vector(x, y) - 创建二维向量
  • scCircle(圆心, 半径) - 创建圆形
  • scRect(左上角, 右下角) - 创建矩形
  • scLine(起点, 终点) - 创建直线
  • ReportMessage(消息, 类型, 是否禁止弹窗) - 消息报告
  • GeMsgReportType - 消息类型枚举

Limit_Int / Limit_Double - 上下限设置

用于设置工具的上下限参数（如 Blob 工具的总像素数上限）

示例代码

# 设置整型上下限
GvTool.SetToolData("Blob工具_001.总像素数上限", GvVisionAssembly.Limit_Int(1000))

# 设置浮点型上下限
GvTool.SetToolData("找线工具_003.直线变化角度上限", GvVisionAssembly.Limit_Double(10.0))

sc2Vector - 二维向量

功能：表示二维向量/坐标点，用于圆心、直线端点、位置坐标等。

方法列表

  • sc2Vector(x, y) - 构造函数
  • GetX() - 获取 X 坐标
  • GetY() - 获取 Y 坐标
  • SetX(值) - 设置 X 坐标
  • SetY(值) - 设置 Y 坐标

构造函数

# 默认构造 (0, 0)
vec = GvVisionAssembly.sc2Vector()

# 指定坐标
vec = GvVisionAssembly.sc2Vector(100, 200)

获取/设置坐标

center = GvTool.GetToolData("找圆工具_001.圆心")

# 获取坐标
x = center.GetX()
y = center.GetY()

# 设置坐标
center.SetX(100)
center.SetY(200)

向量运算

a = GvVisionAssembly.sc2Vector(10, 20)
b = GvVisionAssembly.sc2Vector(5, 10)

c = a + b  # 结果: (15, 30)
d = a - b  # 结果: (5, 10)

scCircle - 圆形

功能：表示圆形，用于创建显示圆形或处理圆形结果。

方法列表

  • scCircle(圆心, 半径) - 构造函数
  • GetCenter() - 获取圆心 (sc2Vector)
  • GetRadius() - 获取半径 (float)
  • SetCenter(sc2Vector) - 设置圆心
  • SetRadius(float) - 设置半径

构造函数

center = GvVisionAssembly.sc2Vector(100, 200)
circle = GvVisionAssembly.scCircle(center, 50)  # 圆心(100,200)，半径50

获取圆的属性

# 从工具获取圆形数据
circle = GvTool.GetToolData("找圆工具_001.圆结果")

# 获取属性
center = circle.GetCenter()
x = center.GetX()
y = center.GetY()
r = circle.GetRadius()

scLine - 直线

构造函数

start = GvVisionAssembly.sc2Vector(0, 0)
end = GvVisionAssembly.sc2Vector(100, 100)
line = GvVisionAssembly.scLine(start, end)

scRect - 矩形

构造函数

top_left = GvVisionAssembly.sc2Vector(10, 10)
bottom_right = GvVisionAssembly.sc2Vector(200, 150)
rect = GvVisionAssembly.scRect(top_left, bottom_right)

GsScriptGuiStyle - GUI 样式

功能：设置 GUI 的显示样式，包括颜色、线型、字体等。

属性列表

属性 类型 说明 示例
bVisible bool 是否可见 True / False
nLineStyle int 线型 (0=实线, 1=虚线, 2=点线, 3=点划线, 4=双点划线) 0
nLineWidth int 线宽 (1-15) 2
clrLineColor list 颜色 [R, G, B] [255, 0, 0]
bLabelVisible bool 标签是否可见 True / False
strLabel string 标签内容 "标签"
strLabelFont string 标签字体 "Arial"
lFontSize int 字体大小 (8-360) 20

示例代码

guiStyle = GvVisionAssembly.GsScriptGuiStyle()

guiStyle.clrLineColor = [255, 0, 0]  # 红色

ReportMessage - 消息报告

函数

GvVisionAssembly.ReportMessage(消息, 类型, 是否禁止弹窗)
参数 类型 说明
消息 string 要报告的消息内容
类型 GeMsgReportType 消息类型
是否禁止弹窗 bool True=禁止弹窗, False=允许弹窗

GeMsgReportType - 消息类型

类型 说明
eMRTFatalErr 致命错误
eMRTErr 错误
eMRTWarn 警告
eMRTInfo 信息

示例代码

msg_type = GvVisionAssembly.GeMsgReportType

# 报告错误

# 报告警告
GvVisionAssembly.ReportMessage("半径过小", msg_type.eMRTWarn, False)

# 报告信息
GvVisionAssembly.ReportMessage("处理完成", msg_type.eMRTInfo, False)

示例：获取找圆结果并显示

from ScImageShow import ScImageShow

pos_i = int(GvVar.GetVar("#i"))

center = GvTool.GetToolData("找圆工具_001.圆心")

示例：显示多个圆

# 假设使用多圆多线查找工具
circles = GvTool.GetToolData("多圆多线查找工具_001.圆结果数组")
count = len(circles)

gui_style = GvVisionAssembly.GsScriptGuiStyle()
gui_style.clrLineColor = [0, 255, 0]
gui_style.lFontSize = 25

gui_array = GvVisionAssembly.GcScriptGuiArray()

for i in range(count):
    circle = circles[i]
    center = circle.GetCenter()
    r = circle.GetRadius()

    # 添加圆形
    guiCircle = GvVisionAssembly.GsScriptGuiCircle()
    guiCircle.circle = GvVisionAssembly.scCircle(center, r)
    guiCircle.sScriptGuiStyle = gui_style
    gui_array.Add(guiCircle)

    # 添加文本标签
    gui_text = GvVisionAssembly.GsScriptGuiText()
    gui_text.strText = f"圆{i+1}"
    gui_text.posX = center.GetX()
    gui_text.posY = center.GetY() - 30
    gui_text.sScriptGuiStyle = gui_style
    gui_array.Add(gui_text)

GvGuiDataAgent.SetGraphicDisplay("View-1", gui_array)

示例：数组解析工具

关键点：数组解析工具输出的是数组，需要用 GetTrans() 方法获取。 
# 获取数组解析工具的输出数据
result = GvTool.GetToolData("数组解析工具_001.输出数据")

# 使用 GetTrans() 方法获取数组
data = result.GetTrans()

# 访问数组元素
x = data[0]   # X坐标
y = data[1]   # Y坐标
r = data[2]   # 半径（如果是圆的话）

print(">>>>>:", x, y, r)
注意：GetTrans() 返回的是数组，可以直接用索引访问。
